# Магазин электроники (Django Project)

## Суть приложения
Данное приложение является учебным проектом (лабораторная работа №16). 
Оно демонстрирует базовую настройку маршрутизации (URL), представлений (Views).

## Список страниц
1. **Главная (`/`)**: Меню навигации с кнопками перехода.
2. **Об авторе (`/author/`)**: Информация об исполнителе работы 
3. **О магазине (`/about/`)**: Описание темы лабораторной работы
